#  Process Inspector (Linux)

A Linux-based Process Inspector built in C that reads the `/proc` filesystem to display process information and memory layout.

---

## Features

-  Inspect process details using PID  
-  Display memory layout (heap, stack, text, data, etc.)  
-  Analyze process state, threads, and name  
-  Categorizes memory sections (HEAP, STACK, TEXT, DATA, etc.)  

---

## Technologies Used

- C Programming  
- Linux System Programming  
- `/proc` File System
